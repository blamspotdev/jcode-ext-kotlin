# J Code — Kotlin Language Pack

A JCode **language** extension for Kotlin: syntax coloring and keyword/type
highlighting, as-you-type completions with snippet helpers, and a best-effort
formatter (`ktlint`/`ktfmt`, run inside the embedded distro). Installed through
the [JCode Marketplace](https://github.com/blamspotdev/j-code-marketplace).

Applies to `.kt` and `.kts` files. It pairs well with the **Android Dev Pack**
for building and running Kotlin/Android projects on device — this pack covers
editing (coloring, completions, snippets, formatting) while the Android Dev Pack
supplies the SDK and toolchain. A Kotlin language server is suggested for richer
analysis.

See [`extension.yaml`](extension.yaml) for the full functional manifest.
